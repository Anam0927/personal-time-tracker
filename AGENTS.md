# time-tracker

This is a **Bun CLI + TUI application** for **personal time tracking on Linux (Omarchy)**.

- **Package manager**: Bun (not npm)
- **Typecheck**: `bun run check`
- **Build**: `bun run build` (compiles to standalone binary)
- **Test**: `bun run test` (automatically includes the `--isolate` flag)
